# element-curd-template

> 基于element ui的curd模板

## 构建步骤

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

## 主要依赖

- vue
- vue-router
- vuex
- axios
- less
- mockjs
- normalize.css
- element-ui

## 目录说明

```bash
.
├── App.vue       根实例
├── api           HTTP API
├── components    组件
├── config        配置
├── main.js       入口js
├── mock          mock请求
├── models        模型
├── router        路由配置
├── store         全局状态管理
├── styles        全局样式
├── utils         工具
└── views         页面视图
```
